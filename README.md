# 🎓 Career Guidance Chatbot

A simple AI-inspired Career Guidance Chatbot built using **Python and Streamlit**.  
The chatbot helps users explore different career paths by providing basic information about fields such as medicine, engineering, science, law, arts, and gaming.

## 🚀 Features

- Interactive chatbot interface
- Career information assistance
- Streamlit-based web application
- Session-based chat history
- Simple natural language keyword matching

## 🛠️ Tech Stack

- Python
- Streamlit
